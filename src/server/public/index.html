📁 What is the Workspace?

NOVA's workspace is a folder on your computer where it stores everything that makes it yours:

  • SOUL.md — NOVA's personality and core values
  • USER.md — what NOVA knows about you
  • MEMORY.md — index of personal memories
  • skills/ — custom skills and automations
  • memory/ — daily notes and reflections

NOVA reads and writes these files automatically as you chat. You can also edit them directly in the Projects panel.

Absolute path. Must exist on disk. Move this folder if you want to back up or sync NOVA's "soul".
